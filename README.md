# Nombre del grupo: Aplicación móvil de transporte para univesitarios
##Curso: Herramientas de Desarrollo
###Descripción: El presente proyecto consiste en el desarrollo de una aplicación móvil inspirada en plataformas de movilidad como Uber, orientada exclusivamente a la comunidad universitaria. La aplicación busca ofrecer un servicio de transporte seguro, económico y confiable mediante la restricción de acceso a usuarios que posean un correo institucional válido de sus respectivas universidades.
